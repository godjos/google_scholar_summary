# Google Scholar 邮件通知处理器项目开发计划

## 1. 项目概述

### 1.1 项目目标

自动化处理 Google Scholar Alerts 邮件，使用大模型 API 对论文进行分析和摘要，提高学术研究效率。

### 1.2 当前状态

- [x] 基础功能已完成
- [x] 支持 QQ 邮箱 IMAP 连接
- [x] 支持大模型 API 调用
- [x] 支持 CSV 结果导出
